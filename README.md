# File Organizer

A Python script to automatically organize files in a directory based on their file types.

## Features

- Organizes files into subdirectories based on their extensions (e.g., images, documents, videos, etc.).
- Monitors the specified directory for changes and reorganizes files as needed.
- Easily customizable to add more file types and categories.

## Requirements

- Python 3.x
- watchdog library
